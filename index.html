1 终端

  1. gnome-terminal : Gnome环境下的终端程序
  2. konsole :KDE的终端程序
  3. urxvt :轻量级高定制终端
  4. yakuake :拉幕式终端，不用总是切换

2 输入法

  1. ibus : 个人觉得比scim好用。
  2. fcitx :小企鹅输入法。

3 编辑器

  1. Emacs :神的编辑器，功能强大，扩展性强。
  2. VIM ：我只用它来编辑一些短小的配置文件。

4 文档办公

  1. WPS Office :在Linux下如果想兼容MSoffice，目前这个可靠些。
  2. Tex/LaTex :专业论文和书籍排版首选工具，Emacs有相应扩展。
  3. org-mode :这个是emacs的扩展，可以用来写wiki,blog,日程安排。
  4. evince :Gnome下PDF阅读工具。
  5. chmsee :chm阅读。
  6. GoldenDict :强大的翻译软件。

5 网络应用

  1. pidgin :多协议，GTK。
  2. Web QQ :少用QQ吧，如果要用，用这个就行了，少折腾。
  3. iptux :局域网信使，可以与飞鸽，飞秋等通信。
  4. xchat :IRC聊天软件。
  5. Firefox :速度一般，插件多。
  6. Chrome :有linux版，速度快。
  7. w3m :字符界面浏览器。
  8. qterm :上BBS。当然你也可以用终端。

6 上传下载

  1. lftp :命令行ftp工具，很好用。
  2. deluge :PyGTK, BT客户端。
  3. wget :命令行下载工具。

7 图形图像

  1. MetaPost :描述性绘图工具。
  2. Dio ：流程图。
  3. gThumb :图片浏览软件。

8 多媒体

  1. xmms :支持mp3,ogg等，Emacs下可扩展。
  2. audacious :支持无损。
  3. mplayer :支持全面，字幕处理强悍。
  4. VLC :貌似这个现在用的人多，我不怎么用。
